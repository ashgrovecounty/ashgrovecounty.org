# Ashgrove County Sheriff's Office — Official Website

Official public-facing website for the Ashgrove County Sheriff's Office, Ashgrove, Wyoming. Established 1891.

---

## About

This repository hosts the static website for the ACSO, providing public access to:

- Department news and press releases
- Missing persons registry
- Department information and staff directory
- Public safety resources and contact information

The site is built with plain HTML, CSS, and JavaScript. No frameworks, no dependencies.
